2. 결제 처리 — 진짜 돈을 받을거냐 fake 랜딩의 유일한 갈림길

A안 · 실결제 → 일정 조율 실패 시 환불

MoMo/ZaloPay QR로 진짜 ₫90k 받는다. 결제자에게는 "48시간 안에 Zalo로 일정 연락" → 수업을 진짜 해주거나, 못 맞추면 환불.

✓ 진짜 수요 — 카드 꺼낸 사람만 카운트
✓ 결제자와 대화 가능 — 왜 샀는지 인터뷰
✗ 수취 계좌·환불 운영 필요

B안 · 결제 버튼까지만 (페이크도어)

"결제하기" 누르면 "오픈 준비 중 — 번호 남기면 1순위 안내" 화면. 돈은 안 받는다.

✓ 운영 0 — 계좌·환불 없음
✗ 전환율 뻥튀기 — 버튼 클릭 ≠ 결제. 실제보다 2~5배 높게 나옴
✗ 광고 태우고 얻는 게 번호뿐
