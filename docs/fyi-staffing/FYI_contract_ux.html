언제

전면 모달(닫기 전 진행 불가). 수수료 5/7% + 핵심 조항 요약 + 전문 링크 + 동의 체크 + 버튼 하나.

체크 후 버튼 = 클릭 동의. 도장·펜 없이 성립. 동의 로그(시각·계정·IP) 저장.
