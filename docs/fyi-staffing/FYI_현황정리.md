# FYI 스태핑 — 현재 상태 정리

## 방향
한국 기업에 콜드로 접근 → 베트남 인재 리모트 매칭. 지금 목표는 바로 매출 잡기.

## 1. 리드 확보 (완료)
- 원티드에서 채용 중인 회사 긁음 → **회사 2,597개, 검증 이메일 1,202개**.
- 직무군별 정리: 개발 463 · 마케팅 542 · 디자인 217 · 운영 812.
- 마스터 시트(구글): https://docs.google.com/spreadsheets/d/1ri2KhR2Y2ulFaHQLJ4Ld33iR79D5ClwEVAOveRHfurA/edit
- 파일: `FYI_staffing_master.xlsx` (6탭). Drive에 드래그하면 구글시트로 변환.
- 기준: "지금 채용 중 + 이메일 있는 회사". 대기업·중복은 발송 전 정리 필요.

## 2. 콜드메일 (초안 있음)
- 먹힌 후크: "베트남 시장", "인건비 1/3". 죽은 후크: "월 99만원/상위5%".
- 4단계 시퀀스(D+0/1/3/4) 초안 잡혀 있음. → 확정만 하면 발송 가능.

## 3. 기업 상품 정책 (결정됨 → `FYI_기업정책.md`)
- 수금: 후보 보려면 먼저 수수료 약정 클릭 동의(= 후보 언락 조건). 큰 건은 오퍼 때 전자서명.
- 상품: 베이직 5% / 프리미엄 7%(헤드헌팅). 착수금 없음, 성공해야 결제. 최소 ₫8M.
- 보증: 입사 60일. 퇴사하면 무료 대체채용, 안 되면 환불.
- 법무: 베트남 준거법. 인력중개 라이선스·세금·개인정보 변호사 확인.
- 리드→공고 전환: 폼 질문 2개로 핫리드만, AM이 공고 대필해 원클릭 게재.
