# FYI 기업 상품 정책 (무대: 베트남)

## 1. 계약 · 수금
지금 문제는 기업이 후보 다 보고 돈을 안 내는 것.

**정책: 후보를 보려면 먼저 수수료 약정에 클릭 동의해야 한다. 동의하면 후보가 열린다.**

- 도장·서명 필요 없음. 클릭 동의 + 기록(누가·언제·IP)만으로 베트남에서 계약 성립.
- 금액 큰 건은 오퍼 단계에서 전자서명(DocuSign) 한 번 더.
- 이유: 너무 세게 묶으면 기업이 안 하고, 약하게 묶으면 지금처럼 못 받는다. 그 중간이 "클릭 동의".

떼먹기 방지:
- 누가 어떤 후보 봤는지 기록 → 우리가 소개했다는 증거
- 입사 때 근로계약서 사본 받기 → 수수료 금액 확정
- 후보한테 어디 입사했는지 확인 → 몰래 뽑은 거 적발
- 안 내면 지연이자 + 세금계산서로 청구

## 2. 법무
- 준거법은 베트남 (지금 약관 그대로). 한국 통신판매업은 무대가 베트남이라 해당 없음.
- 할 일: 인력중개 라이선스 확인 / VAT·FCT 세금 / 개인정보(베트남 PDPD) / 약관 감수. 변호사 1번 확인.

## 3. 상품
- **베이직 5%**: 공고 + ATS + 알아서 들어온 지원자.
- **프리미엄 7%**: + 우리가 직접 인재 찾아줌. 2주 안에 적합 후보 3명 못 주면 5%로 내림.
- 착수금 없음. 성공해야 돈 낸다.
- 수수료 = 계약연봉의 5~7%. 단 너무 낮으면 최소 ₫8M.
- 보증 60일: 60일 안에 퇴사하면 다른 사람 무료로 다시 뽑아줌. 안 되면 전액 환불.
- 입사하면 청구.

## 4. 리드가 공고로 안 넘어오는 이유
리드 7천원은 싼 게 아니라 쉬운 거다. 메타 폼이 전화번호를 자동으로 채워줘서 아무나 낸다.

고칠 것:
- 폼에 질문 2개 넣기 (지금 뭐 뽑아요? 급해요?) → 대충 낸 사람 걸러짐
- 전화 받기 대신 상담 시간 예약 잡기
- 공고는 AM이 대신 써주고, 기업은 "이대로 올리기" 한 번만 누르게
- 폼 내면 바로 핏 후보 3명 보여주기

## 체크리스트
- 법무: 라이선스 / 약관 개정(5·7%, 60일, 클릭동의) / 세금·개인정보
- 개발: 클릭 동의 게이트 / 폼 질문+상담예약 / AM 공고 대필 원클릭 / 최소수수료·청구
- 운영: 60일 대체채용 / 근로계약서 사본 / 후보 입사확인
