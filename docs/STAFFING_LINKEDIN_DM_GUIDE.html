링크드인 DM 광고 세팅

베트남 인재 스태핑 · 대화 광고 · 리드는 공고마감(ggmg.ai.kr)으로

실물

우리가 만드는 광고는 이렇게 생겼다

흐름: DM(짜놓은 멘트) → 버튼 → 자동 채움 신청서 → [전송]. 여기까지 링크드인 안에서 끝나고, 받은 연락처로 우리가 이메일 보내는 순간부터 진짜 대화 시작.

0

시작 전 준비

  • 회사 페이지 + Campaign Manager 계정 (결제 카드 등록까지)
  • 발신자 프로필 정리 — DM 상단에 사진·헤드라인 그대로 노출됨
  • 리드 양식 먼저 만들기 — 자산(Assets) → 리드 양식. 광고 화면에선 못 만든다 (5번 참고)
  • ggmg.ai.kr/privacy 열리는지 확인 — 리드 양식 저장에 필수
  • 리드 담당 정하기 — 알림 자동으로 안 옴. 매일 다운로드 + 1영업일 회신 담당

제목

첫 메시지 (%코드%는 [매크로 추가]로 삽입 — 이름·회사명 자동 치환)
안녕하세요 %FIRSTNAME%님, %COMPANYNAME%처럼 성장 중인 팀은 채용할 자리는 많은데 인건비가 제일 무겁죠. 저희는 한국 기업이 검증된 베트남 인재를 리모트로 쓰도록 연결합니다. 같은 역할 기준 인건비가 1/3 수준이고, 착수금 없이 채용이 성사돼야 수수료를 받습니다. 어떤 부분이 제일 궁금하세요?
버튼 1 → 이어지는 메시지 → [무료 견적 받기]=리드 양식
비용이 얼마나 절감되나요? --- 예를 들어 3년차 백엔드 개발자 기준, 국내 채용 대비 월 고정비가 1/3 수준입니다. 60일 안에 퇴사하면 무료로 대체 인력을 붙여드리고, 안 되면 환불합니다. 회사 상황 기준으로 견적을 바로 계산해드릴까요?
버튼 2 → 이어지는 메시지 → [인재 프로필 보기]=리드 양식
어떤 인재들이 있나요? --- 개발·마케팅·디자인·운영 직군의 실무 검증된 인재풀을 보유하고 있습니다. 상위 대학 출신과 한국 기업 협업 경험자 중심입니다. 어떤 직군을 뽑고 계신지 알려주시면 맞는 프로필을 보여드릴게요.
버튼 3 (거절 탈출구) → 이어지는 메시지
지금은 괜찮습니다 --- 알겠습니다. 나중에 인력 비용을 고민하실 때 편하게 찾아주세요. 좋은 하루 되세요!

③에서 리드 양식 "스태핑_견적문의" 선택 → 빨간 필수 에러 사라짐. 배너·바닥글은 비워두고 진행.
