링크드인 DM 광고 세팅

채용이 비싸고 어렵다면 이런 방법도 있어요. 대화 광고로 받은 리드는 공고마감(ggmg.ai.kr)으로.

실물

우리가 만드는 광고는 이렇게 생겼다

흐름: DM(미리 써둔 멘트) → 버튼 → 자동으로 채워진 신청서 → [전송]. 여기까지 링크드인 안에서 끝나고, 받은 연락처로 우리가 이메일 보내는 순간부터 진짜 대화 시작.

0

시작 전 준비

  • 회사 페이지 + Campaign Manager 계정 (결제 카드 등록까지)
  • 발신자 프로필 정리 — DM 상단에 사진·헤드라인 그대로 노출됨
  • 리드 양식 먼저 만들기 — 자산(Assets) → 리드 양식. 광고 화면에선 못 만든다 (5번 참고)
  • ggmg.ai.kr/privacy 열리는지 확인 — 리드 양식 저장에 필수
  • 리드 담당 정하기 — 알림 자동으로 안 옴. 매일 다운로드 + 1영업일 회신 담당

톤 원칙 (문구 전부 여기에 맞춘다)
① 아픔으로 연다 — 채용이 비싸고 어렵다는 현실을 건드리고 "이런 방법도 있어요"로 부드럽게 대안 제시. ② 니즈 먼저 — 가격·상품 들이밀기 전에 지금 채용 중인 자리(JD)부터 묻는다. ③ 양식 제출 후엔 AM 담당자가 직접 연락해 JD 확인하고 맞는 검증 인재 안내(이력서 자동발송 아님). ④ 인도/베트남·3분의 1 비용은 DM 앞단(제목·첫 메시지)엔 안 꺼내고, 관심 확인된 리드 양식부터 공개. ⑤ 끝까지 "부담 없이 확인만 해보셔도 좋아요" 톤, 밀어붙이지 않는다.

제목 (미리보기 한 줄에 뜨는 사실상의 후크)
채용, 너무 비싸고 어렵다면 이런 방법도 있어요
첫 메시지 (%코드%는 [매크로 추가]로 삽입 — 이름·회사명 자동 치환)
안녕하세요 %FIRSTNAME%님, 요즘 사람 한 명 뽑기, 비싸고 어렵죠. 혹시 %COMPANYNAME%에서 지금 채용 중이신 자리가 있으신가요? 있다면 그 부담을 크게 덜어드릴 방법이 있어서요. 어떤 자리인지만 알려주시면 맞는 인재를 찾아 안내해드릴게요. 부담 없이 확인만 해보셔도 좋아요.
버튼 1 → 이어지는 메시지 → [채용 중인 자리 알려주기]=리드 양식
지금 채용 중인 자리가 있어요 --- 좋아요. 어떤 자리를 채용 중이신지 남겨주시면, 담당 매니저가 바로 연락드려 확인 후 딱 맞는 검증 인재를 안내해드릴게요. 부담 없이 편하게 이어가시면 됩니다.
버튼 2 → 이어지는 메시지 → [어떤 방법인지 궁금해요]=리드 양식
어떤 방법인지 궁금해요 --- 정규직 채용이라는 무거운 과정 없이, 검증된 인재를 리모트로 빠르게 연결해 드려요. 채용 중이신 자리만 알려주시면 담당 매니저가 딱 맞는 인재를 찾아 안내해드릴게요. 부담 없이 확인만 해보셔도 좋아요.
버튼 3 (거절 탈출구) → 이어지는 메시지
지금은 괜찮아요 --- 알겠습니다. 나중에 사람이 필요하실 때 편하게 찾아주세요. 좋은 하루 되세요!

③에서 리드 양식 "스태핑_견적문의" 선택 → 빨간 필수 에러 사라짐. 배너·바닥글은 비워두고 진행.
